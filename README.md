## What

## Installation

```bash
yarn add hardhat-change-network
```

Import the plugin in your `hardhat.config.js`:

```js
require("hardhat-change-network");
```

Or if you are using TypeScript, in your `hardhat.config.ts`:

```ts
import "hardhat-change-network";
```
